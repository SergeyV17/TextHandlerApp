				Описание работы приложения:

Программа обработчик текста

Для начала добавьте файлы и нажмите кнопку «Продолжить».

Для более подробной настройки используйте панель параметров, которая включает в себя:
- Удаление слов, содержащих меньше указанного количества символов
- удаление знаков препинания

Эта программа работает только с форматом файла .txt.

P.S.
— Если хотя бы один символ в наборе символов не является буквой (кроме пунктуации), 
то данный набор символов не является словом, обработка опускается, полученный набор символов 
добавляется в строку.
— Удалённые слова заменяются на пробелы

			Программные средства необходимые для запуска и работы

1) IDE Visual Studio версия .NET Framework 4.7.2
2) NuGet package System.Windows.Interactivity.WPF
3) NuGet package WindowsAPICodePack-Shell
